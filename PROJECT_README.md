# barbershop-saas

> Мини-SaaS для барбершопов: лендинг + личный кабинет владельца, Next.js + Tailwind + Drizzle + NextAuth, деплой Vercel

**Template**: `saas-nextjs`  **Created**: 2026-03-10

## Quick start

```bash
cd /root/projects/barbershop-saas
pnpm install
cp .env.example .env.local
pnpm db:setup && pnpm db:migrate
pnpm dev
```
